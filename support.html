Common Questions

I can't sign in

Make sure you're using the same sign-in method you signed up with (Apple or email). If you signed up with Apple, you won't have a password — tap "Sign in with Apple" again. For email accounts, use the "Forgot password?" link on the sign-in screen, or send us a message above and we'll help.

How do I delete my account?

Open NomadAI → Settings → Account → Delete Account. This permanently removes your profile, saved spots, trips, reviews, and Passport stamps from every device. It cannot be undone.

The AI search didn't return what I expected

AI search results are generated fresh each time, so the same query can return different campsites on different runs. Try rephrasing — adding a region ("near Bend, OR"), a constraint ("with showers" or "BLM only"), or a timeframe ("this weekend") usually sharpens the results. If a result looks wrong, send us a message with the query and we'll investigate.

A campsite's details are wrong or outdated

Campsite data is aggregated from federal reservation systems, public-land databases, and community-mapped overland sources, then refreshed weekly. If you see something inaccurate — wrong coordinates, closed access road, broken amenities listing — let us know and we'll correct it.

The app crashed

NomadAI captures crash reports automatically (anonymized — no personal data attached). If it keeps happening, send us a message describing what you were doing and we'll dig in. Restarting the app or reinstalling usually resolves transient issues.

Feedback, feature requests, or anything else

Use the form above, or email support@nomadai.us. The app also has a Feedback button inside the Help section that goes to the same place.
